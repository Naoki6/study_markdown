## テキストの装飾
**太字**: `**太字**` → **太字**

*斜体*: `*斜体*` → *斜体*

~~取り消し線~~: `~~取り消し線~~` → ~~取り消し線~~

**斜体 + 太字**: `***斜体 + 太字***` → ***斜体 + 太字***
